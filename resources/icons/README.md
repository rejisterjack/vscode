# FewStepAway Icons

This directory contains the FewStepAway application icons.

## Current Status

Using VS Code icons as placeholders until custom FewStepAway icons are designed.

## Icon Requirements

### macOS
- Format: .icns
- Location: `resources/darwin/fewstepaway.icns`
- Sizes: 16x16, 32x32, 128x128, 256x256, 512x512, 1024x1024

### Windows
- Format: .ico
- Location: `resources/win32/fewstepaway.ico`
- Sizes: 16x16, 32x32, 48x48, 256x256

### Linux
- Format: .png
- Location: `resources/linux/fewstepaway.png`
- Sizes: 16x16, 32x32, 48x48, 64x64, 128x128, 256x256, 512x512

### Server
- Format: .png
- Location: `resources/server/fewstepaway-192.png`, `fewstepaway-512.png`

## Design Guidelines

- Modern, flat design
- AI/coding theme
- Distinctive from VS Code
- Works on dark and light backgrounds
- Scalable from 16px to 1024px
