# Reference Repositories

This folder contains reference implementations and resources for the FewStepAway AI-native code editor project.

## AI Coding Assistants

### Cline (cline)
**Repository**: https://github.com/cline/cline
**Description**: Popular VS Code extension for AI coding assistance
**Key Features to Study**:
- Chat interface integration
- File operations and context management
- Multi-provider AI support (OpenAI, Anthropic, etc.)
- Tool calling implementation
- Terminal integration

### Roo-Code (RooVetGit/Roo-Code)
**Repository**: https://github.com/RooVetGit/Roo-Code
**Description**: Advanced AI coding assistant (formerly Roo Cline)
**Key Features to Study**:
- Enhanced chat capabilities
- Mode-based AI interactions
- Advanced context management
- Code editing and refactoring
- Browser automation capabilities

### KiloCode (Kilo-Org/kilocode)
**Repository**: https://github.com/Kilo-Org/kilocode
**Description**: AI coding assistant extension
**Key Features to Study**:
- AI-assisted coding workflows
- Extension architecture patterns
- Integration with VS Code APIs
- AI provider management

## Notes

- All major reference repositories have been cloned
- These references provide implementation patterns for AI-native features
- Study the architecture, not copy code directly
- Focus on integration patterns, UI components, and AI provider abstractions

## Usage

When implementing features in FewStepAway, refer to these repositories for:
1. **Chat Interface**: How to integrate conversational AI into the editor
2. **Ghost Text**: Inline AI suggestions and completions
3. **Context Management**: Providing codebase context to AI models
4. **Tool Calling**: Code execution, file operations, terminal commands
5. **Provider Abstraction**: Supporting multiple AI providers (OpenAI, Anthropic, Google, etc.)
6. **MCP Integration**: Standardized tool interfaces for AI agents
